# Правила при работе с кодом

- Перед изменением слоя сверься с `memory-bank/architecture/overview.md`:
  зависимости направлены только внутрь (к домену), домен не знает об инфраструктуре.
- Новые публичные интерфейсы/контракты — отрази в `memory-bank/architecture/patterns.md`,
  если вводишь новый паттерн.
- Изменил поведение, описанное в `memory-bank/domain/business-rules.md`, —
  обнови документ в том же PR.
