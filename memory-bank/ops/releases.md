# Релизы {{PROJECT_NAME}}

Как изменения попадают в prod и как их откатить. Ветки и правила коммитов —
[../architecture/git-workflow.md](../architecture/git-workflow.md);
окружения — [environments.md](environments.md). Срочные фиксы идут мимо
этого процесса — по [../workflows/hotfix.md](../workflows/hotfix.md).

## Версионирование

- Схема: **SemVer** `MAJOR.MINOR.PATCH` — TODO(template): подтвердить или заменить
  (CalVer, сквозной build number) и записать, что именно означает каждая часть
  для продукта (что считается breaking change наружу).
- Версия фиксируется: TODO(template): git-тег `vX.Y.Z` / поле в файле проекта / и то и другое.
- Changelog: TODO(template): ведётся ли, где, генерируется ли из conventional commits.

Пример (замените своим): SemVer; MAJOR — несовместимое изменение публичного API,
MINOR — новая функциональность, PATCH — фиксы. Тег `v2.4.1` ставит CI при релизе.

1. Убедиться, что в main всё, что входит в релиз, и CI зелёный.
2. Пройти чек-лист перед релизом (ниже).
3. Создать релиз: TODO(template): команда/кнопка (тег, релизная ветка, GitHub Release).
4. Деплой на **stage**: TODO(template): как запускается; прогнать регресс/смоук.
5. Получить подтверждение: TODO(template): кто аппрувит выход на prod.
6. Деплой на **prod**: TODO(template): как запускается; ручной шаг или авто после stage.
7. Пост-проверка prod (15–30 минут): смоук-сценарий, дашборды и алерты —
   [observability.md](observability.md).
8. Объявить о релизе: TODO(template): канал/формат.

Агент шаги 3–7 сам не выполняет — готовит изменения и чек-лист, запуск за человеком
(см. ограничения в [environments.md](environments.md)).

## Чек-лист перед релизом

- [ ] CI зелёный на релизном коммите; тесты не скипнуты.
- [ ] Миграции БД обратимы или есть план отката данных.
- [ ] Новые переменные окружения заведены во всех окружениях
  ([configuration.md](configuration.md)).
- [ ] Breaking changes API согласованы с потребителями — TODO(template): с кем.
- [ ] Версия и changelog обновлены.
- [ ] Известно, какой релиз сейчас в prod (точка отката).
